# Peliculas-del-Estudio-Ghibli-II
Lo que he hecho en este proyecto:
- Conexión automática: He configurado la página para que llame a la API de Ghibli en cuanto se abre, así las películas salen solas sin tener que darle a ningún botón.
- Uso de la API real: He conectado el código con la dirección https://ghibliapi.vercel.app/films para traer los datos actualizados.
- Buscador en tiempo real: He añadido una barra de búsqueda que filtra las películas por el nombre mientras vas escribiendo.
